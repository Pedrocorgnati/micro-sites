Site lento perde clientes — e os dados são claros sobre isso. Segundo o Google, **53% dos visitantes mobile abandonam uma página que demora mais de 3 segundos para carregar**. Se seu site demora 6 segundos, você perdeu mais da metade das visitas antes mesmo de mostrar o que faz.

## O custo real de um segundo de lentidão

A Amazon calculou que cada 100 milissegundos de lentidão reduz as vendas em 1%. Para uma empresa de e-commerce, isso é catastrófico. Para um prestador de serviços ou loja local, o impacto proporcional é semelhante.

Imagine que seu site recebe 500 visitas por mês de usuários mobile. Se ele demora 6 segundos para carregar:
- 265 visitantes abandonam antes de ver qualquer coisa (53%)
- Você converte talvez 2% dos que ficam: ~4-5 leads por mês

Agora imagine o mesmo site carregando em 1,5 segundos:
- Taxa de abandono cai para menos de 10%: 450 visitantes ficam
- Os mesmos 2% de conversão: ~9 leads por mês

Apenas por melhorar a velocidade, você pode dobrar a geração de leads sem nenhum outro investimento em marketing.

## Por que sites ficam lentos

Existem várias causas comuns:

**Imagens não otimizadas:** Uma foto de 5MB carregada diretamente no site é o erro mais comum. Imagens devem ser comprimidas e no formato certo (WebP para web moderna).

**Plugins e scripts desnecessários:** Cada plugin instalado adiciona peso. Sites WordPress antigos frequentemente têm dezenas de plugins ativos, muitos sem uso.

**Hospedagem inadequada:** Servidor compartilhado com recursos limitados é ótimo para economizar, péssimo para performance.

**Código não otimizado:** JavaScript e CSS não minificados, sem compressão, carregados de forma bloqueante.

**Sem CDN:** Sem rede de entrega de conteúdo, os arquivos vêm de um servidor único, mais longe do visitante.

## Como verificar a velocidade do seu site

Use o Google PageSpeed Insights (gratuito): basta digitar a URL e você recebe uma nota de 0 a 100 para mobile e desktop, com diagnóstico dos problemas.

Pontuações de referência:
- **90-100:** Excelente
- **50-89:** Precisa de melhorias
- **0-49:** Crítico — você está perdendo clientes agora

## Velocidade e SEO: a conexão direta

Desde 2021, o Google usa Core Web Vitals como fator de rankeamento. Esses são indicadores de experiência do usuário, e velocidade é o principal deles:

- **LCP (Largest Contentful Paint):** Quanto tempo até o maior elemento visível carregar. Meta: < 2,5 segundos
- **FID/INP (Interaction to Next Paint):** Tempo de resposta a interações. Meta: < 200ms
- **CLS (Cumulative Layout Shift):** Estabilidade visual da página. Meta: < 0,1

Um site com métricas ruins nesses indicadores está em desvantagem no Google, independente de qualidade de conteúdo ou backlinks.

## Como resolver: o que fazer com um site lento

Se seu site tem problemas de velocidade, as soluções variam de simples a complexas:

**Correções rápidas (você mesmo pode fazer):**
- Comprimir imagens com ferramentas como TinyPNG
- Instalar plugin de cache (se usar WordPress)
- Remover plugins não utilizados

**Correções intermediárias (com ajuda técnica):**
- Migrar para hospedagem melhor
- Implementar CDN
- Minificar CSS e JavaScript

**Modernização completa (para sites muito desatualizados):**
- Reconstruir com stack moderna e performática
- Implementar otimizações de imagem automáticas
- Migrar para infraestrutura adequada

Para sites com mais de 3-4 anos, a modernização completa frequentemente é mais eficiente do que tentar corrigir um por um os problemas de um sistema desatualizado.

## Além da velocidade: outros problemas de sites antigos

Velocidade é um sintoma, mas sites antigos têm outros problemas que também custam clientes:

- **Não funciona em smartphones:** Layout quebrado, texto ilegível, botões não clicáveis
- **Design desatualizado:** Credibilidade comprometida pela aparência antiga
- **Formulários quebrados:** Você perde leads sem saber
- **Sem HTTPS:** Navegadores mostram aviso de "site não seguro", afastando visitantes

## Conclusão

Site lento é um problema de negócio, não apenas de tecnologia. Cada segundo de carregamento custa visitas, leads e faturamento.
