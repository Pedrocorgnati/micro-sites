## O dilema de quem quer ter um site

Você precisa de um site. Mas tem duas opções bem diferentes pela frente:

1. **Plataforma pronta** (WordPress, Wix, Squarespace, Shopify) — cria você mesmo ou contrata alguém para configurar
2. **Desenvolvimento sob medida** — contrata uma agência ou freelancer para criar do zero

A decisão errada pode custar tempo e dinheiro. Este guia te ajuda a escolher certo.

---

## Plataformas prontas: quando fazem sentido

### WordPress (autohospedado)

**Para quem:** Blogs, sites de conteúdo, pequenas lojas, sites institucionais.

**Vantagens:**
- Enorme ecossistema de plugins (55.000+)
- CMS completo para gerenciar conteúdo
- Hosting próprio = controle total
- Custo inicial baixo (R$ 500–3.000 para montar)

**Desvantagens:**
- Requer manutenção constante (updates de segurança)
- Performance exige otimização técnica
- Plugins ruins podem criar vulnerabilidades

**Custo real:** R$ 500–5.000 para montar + R$ 300–1.500/ano de manutenção

---

### Wix / Squarespace

**Para quem:** Autônomos, profissionais liberais, sites simples e portfólios.

**Vantagens:**
- Arrasta e solta — sem código
- Hosting incluso
- Suporte da plataforma

**Desvantagens:**
- Você aluga, não possui o site
- Difícil migrar depois
- Personalização limitada
- SEO mediocre comparado a código próprio

**Custo real:** R$ 80–250/mês (R$ 960–3.000/ano)

---

### Shopify

**Para quem:** E-commerce com volume de vendas relevante.

**Vantagens:**
- Melhor plataforma de e-commerce do mercado
- Pagamentos, estoque, relatórios nativos
- App Store com 8.000+ integrações

**Desvantagens:**
- Caro para quem está começando (US$ 39–399/mês)
- Taxa de transação adicional

**Custo real:** R$ 200–2.000/mês

---

## Desenvolvimento sob medida: quando vale o investimento

### Quando plataforma NÃO é suficiente

Você precisa de desenvolvimento customizado quando:

| Situação | Por que plataforma não resolve |
|----------|-------------------------------|
| Lógica de negócio complexa | Plugins não cobrem seu caso de uso |
| Integrações proprietárias | API personalizada com seu ERP/CRM |
| Performance crítica | INP < 200ms, LCP < 1.5s |
| Múltiplos sites com mesma base | Motor gerador de sites (como Micro Sites) |
| Conformidade regulatória | LGPD, PCI-DSS com controle total do código |

### Custo de desenvolvimento sob medida

| Tipo | Freelancer | Agência pequena | Agência média |
|------|-----------|-----------------|---------------|
| Landing page | R$ 1.500–4.000 | R$ 3.000–8.000 | R$ 8.000–15.000 |
| Site institucional | R$ 3.000–8.000 | R$ 6.000–15.000 | R$ 15.000–35.000 |
| E-commerce | R$ 8.000–20.000 | R$ 15.000–40.000 | R$ 40.000–100.000 |
| Sistema web | R$ 15.000–50.000 | R$ 30.000–80.000 | R$ 80.000–200.000+ |

## A regra de ouro para decidir

**Use plataforma se:**
- Seu orçamento é < R$ 5.000
- Você quer gerenciar o conteúdo você mesmo
- Não tem regras de negócio complexas
- Está validando uma ideia

**Invista em desenvolvimento se:**
- O site é parte crítica do seu produto
- Você precisa de integrações customizadas
- Performance e SEO são diferenciais competitivos
- Vai escalar para múltiplos sites ou mercados
