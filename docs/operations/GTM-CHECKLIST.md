# GTM Checklist

## Pós-deploy (por site)

- [ ] Submissão manual no Google Search Console (sitemap.xml)
- [ ] Verificação de indexação: `site:<dominio>` no Google 48h após deploy
- [ ] Lighthouse CI passou com score mobile >=0.95
- [ ] Formulários de contato testados
- [ ] Redirects 301 configurados (.htaccess)
- [ ] Sitemap + robots.txt acessíveis

## Divulgação

- [ ] Compartilhar sites D (ferramentas) em LinkedIn com contexto — sem spam
- [ ] Posts da SystemForge destacando F01/F02 (comparativos)
- [ ] Material de apoio em propostas comerciais (link direto)
- [ ] Casos em comunidades relevantes (somente quando genuinamente útil)

## Monitoramento inicial (primeiros 2 meses)

- [ ] Check semanal de indexação via GSC
- [ ] Alerta manual se 0 impressões em 2 semanas → revisar SEO/conteúdo
- [ ] Lighthouse mensal via cron + leitura do report
