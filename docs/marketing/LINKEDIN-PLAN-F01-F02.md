## Estrategia

Os blogs F01 (desenvolvimento web) e F02 (marketing digital) tem
audiencia complementar mas distinta:

- **F01 audience:** dev, tech lead, CTO de PME, freelancer tech
- **F02 audience:** marketing manager, dono de PME B2B, agencia

LinkedIn e canal natural para top-of-funnel desses dois publicos.
Cada post LinkedIn:
- Funciona como teaser de artigo do blog (inbound)
- Posiciona Pedro como autoridade (Trust)
- Convida para acao curta (curtir/comentar/visitar -> CTA)

## Formato padrao

**Texto longo (300-500 palavras):**
1. Hook (1 linha provocativa, fim com `:`)
2. Contexto (2-3 paragrafos)
3. 3 takeaways numerados
4. Conclusao curta (1 linha)
5. CTA + link blog
6. Hashtags (5-7 max)

**Carrossel (8-12 slides PDF):**
- Slide 1: titulo + visual forte
- Slide 2-9: 1 ideia por slide
- Slide 10: CTA
- Slide 11-12: bio Pedro + link

## CTA padrao

- "Comente abaixo {pergunta}" -> engagement
- "Salve para ler depois" -> reach
- "Link no primeiro comentario" -> blog click
