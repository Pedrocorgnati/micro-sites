# Templates

Indice de templates reutilizaveis para micro-sites.

| Template | Uso | Componente base |
|----------|-----|-----------------|
| landing/ | Sites Cat. A/B/C/E (institucional, pre-SaaS, nicho) | HeroSection + F01/B01/A01 |
| blog/ | Layout de blog com BlogList/BlogSearch | BlogList, BlogSearch |
| calculator/ | Sites Cat. D (ferramentas interativas) | Calculator, D05Checklist |

Cada subpasta contém READMEs documentando: quando usar, props, slots.
Os componentes continuam residindo em `src/components/`; este diretorio é o índice de templates de alto nível.
