# Template: blog

Uso: páginas `/blog` e `/blog/[slug]` de micro-sites com conteúdo SEO.

Componentes base:
- `src/components/blog/BlogList.tsx`
- `src/components/blog/BlogSearch.tsx`
- `src/app/blog/page.tsx`

Comportamento: empty state automático quando sem artigos; search ativado com >=5 artigos.
